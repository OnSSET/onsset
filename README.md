# Scope

This repository contains the source code of the Open Source Spatial Electrification Tool as seen in the OpenLearn course (2025).
([OnSSET](http://www.onsset.org/)).


The repository also includes sample test files available in ```.\test_data```
and sample output files available in ```.\sample_output```.

## Installation

Install the onsset_env_yml file to be able to run this version.
